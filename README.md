# 🎮 LoL Champions Catalog

Каталог чемпионов League of Legends на Vue 3.

## 🛠️ Технологии
- HTML5
- CSS3 (Flexbox, Grid, адаптив)
- Vue.js 3 (Composition API)
- Data Dragon API (Riot Games)
## ✨ Возможности
- Поиск чемпионов по имени
- Фильтр по ролям (Mage, Assassin, Fighter, Tank, Support, Marksman)
- Просмотр способностей (Q, W, E, R) с описанием
- Адаптивный дизайн
